# AdminSwitch - Project

AdminSwitch is a Next.js application designed to provide a secure admin dashboard and a maintenance mode switch. This project allows administrators to easily manage their application's availability and user access.

## Features

*   **Secure Admin Dashboard:** A dedicated admin dashboard, protected by user authentication, for managing the application.
*   **Maintenance Mode Switch:** A simple toggle to enable or disable maintenance mode, allowing administrators to perform updates or maintenance without disrupting users.
*   **Maintenance Mode Indicator:** A clear visual indicator displayed to users when the application is in maintenance mode.

## Style Guidelines

The project follows a clean and modern design aesthetic with a professional color scheme:

*   **Primary Color:** Dark blue (`#2C3E50`)
*   **Secondary Color:** Light gray (`#ECF0F1`)
*   **Accent Color:** Teal (`#3498DB`)

The layout emphasizes a clean and intuitive dashboard with clear navigation and modern icons.

## Getting Started

This project is built on Next.js. To get started:

1.  **Explore `src/app/page.tsx`:** This is the main entry point for the application's front-end.
2. **Explore `src/hooks/use-admin.tsx`:** This hook determines if a user is an admin.
